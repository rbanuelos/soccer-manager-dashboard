# Soccer Manager Dashboard
A lightweight API for a Soccer Manager Dashboard

### Install
```bash
npm install
```

### Build
```bash
npm run build
```

### Run
```start Mongo DB
docker-compose up -d
```

```bash
npm run start
```

## Run with in development mode with nodemon
```bash
npm run dev
```
